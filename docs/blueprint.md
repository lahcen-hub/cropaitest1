# **App Name**: CropAI

## Core Features:

- Farm Profile: Collect user's profile: role, crops, area, language, and location (GPS or map selection).
- Nearby Resources: Display nearby suppliers, providers, cooperatives, and regional offices based on location.
- Disease/Pest Detection: Analyze uploaded images of plants to identify diseases or pests using a pre-trained image classification model as a tool.
- AI Diagnosis & Treatment: Provide diagnosis and treatment suggestions using OpenAI's ChatGPT API, available in English, French, and Arabic as a tool.
- Crop-Specific Tools: Filter and display relevant tools based on the crops selected in the user's profile.
- Personalized Calendar: Generate personalized irrigation and fertilization schedule as a tool, based on crop, area, and location-specific weather.

## Style Guidelines:

- Primary color: A vibrant green (#90EE90) to represent growth and nature.
- Background color: Light, desaturated green (#F0FAF0) for a calm, natural feel.
- Accent color: Yellow-green (#B0E87D) to highlight important elements.
- Body font: 'PT Sans', a humanist sans-serif, is suited to either headline or body use.
- Use clean and intuitive icons representing different crops, tools, and locations.
- Implement a responsive and intuitive layout that adapts to different screen sizes.
